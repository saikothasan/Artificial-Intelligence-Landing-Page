## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Artificial-Intelligence-Landing-Page-React.git
```

Go to the project directory

```bash
  cd artificial-intelligence-landing-page-react

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
